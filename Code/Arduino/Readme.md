### Note

Make sure that when you write your main code, it is placed in its own folder under **`Arduino/`** with the exact same name as the **`.ino`** file (for example, **`Arduino/MyProject/MyProject.ino`**). The Arduino IDE requires that the folder and the main file have the same name.
