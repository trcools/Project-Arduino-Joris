# Componenten-inventaris — Arduino/Elektronica Project

---

## Gesalvagede HP Pavilion g6 laptop-onderdelen

- CPU-koelventilator (4-draads, PWM) + heatsink/heatpipe
- Moederbord (HP DA0R33MB6E0)
- 2x CD/DVD laser pickup-assemblies (met voice-coil actuators voor focus/tracking)
- DDR3 RAM (Hynix, 2x)
- SATA HDD's (Hitachi 500GB, incl. spindelmotor + voice-coil leeskoparm met neodymium-magneten)
- Synaptics TM2068 touchpad
- Panasonic DVD-RW optisch station (model UJ8B1, laserdiode 300–500 mW @ 650 nm)
- Power button PCB
- Ralink RT5390 wifikaart
- Li-Ion accu (10.8V)

## Arduino starterkit

---

### Basis

- Arduino Uno (kloon)
- Groot breadboard + mini breadboard
- Jumperwires M-M (ribbon + losse)
- Prototype shield
- Breadboard-voedingsmodule (MB102-type, 3.3V/5V)

### Sensoren

- HC-SR04 ultrasone afstandssensor
- DHT11 temperatuur/vochtigheidssensor
- Regendruppel/watersensor
- Geluid/microfoon-sensormodule
- IR-ontvanger + IR-afstandsbediening
- Losse LDR/fotoweerstand-achtige sensor

### Actuatoren & motoren

- SG90 micro-servo + hoorntjes
- Kleine DC-motor + propeller
- 28BYJ-48 stappenmotor + ULN2003-drivermodule
- 5V relaismodule (Songle SRD-05VDC-SL-C)

### Input/output

- 16x2 LCD-scherm + I²C-backpackmodule
- 4x4 matrix keypad
- Analoge joystick-module
- 1-cijfer 7-segment display
- 4-cijfer 7-segment display
- Meerdere tactile push buttons

### Elektronica-onderdelen

- Diverse weerstanden (veel, verschillende waarden)
- Diverse LEDs (meerdere kleuren)
- IC's: SN74HC595 shift register, ULN-serie driver DIP-chip
